## Limitations

- LegacyOps is a **demo/pilot-readiness scaffold**, not a production
  deployment. No real database, no real auth, no real Siebel connection.
- All data is **synthetic**. No real customer data is used.
- The Siebel reference captures are **public URLs**, not downloaded
  images. The ZIP contains the index files only.
- Playwright screenshots are **not part of CI** — they are on-demand via
  `pnpm screenshot:web`.
- The Siebel reference analysis is based on **public documentation and
  community screenshots**, not on access to a real Siebel instance.
