# Siebel Reference — Extracted Index

This directory contains the text files from `siebel_capturas_pack_v2.zip`.
These files are an index of public Siebel CRM screenshot URLs and their
UX analysis descriptions. No binary images are included.

## Files

| File | Purpose |
|---|---|
| `siebel_capturas_descarga_v2.html` | HTML gallery with 16 image references |
| `siebel_capturas_sources_v2.csv` | Structured table: id, tag, density, title, why, image_url, source_url |
| `siebel_capturas_links_v2.txt` | Plain-text list of direct image URLs |
| `README_siebel_capturas_v2.txt` | Pack description |

## Legal note

These files contain **public URLs** to images hosted on third-party sites
(blogger.googleusercontent.com, siebelhub.com, docs.oracle.com, crmcog.com).
No images were downloaded. No proprietary content is reproduced. The files
are used solely as a reference index for UX pattern analysis.
