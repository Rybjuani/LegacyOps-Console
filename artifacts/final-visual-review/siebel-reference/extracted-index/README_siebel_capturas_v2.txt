Pack v2 de capturas densas de Oracle Siebel CRM

Archivos:
- siebel_capturas_descarga_v2.html: galería con previsualización remota y enlaces.
- siebel_capturas_links_v2.txt: lista simple de enlaces directos.
- siebel_capturas_sources_v2.csv: tabla con título, densidad, URL de imagen y fuente.

Uso:
1. Abrir el HTML.
2. Clic en “Abrir / descargar imagen original”.
3. Guardar la imagen desde el navegador.

Nota técnica:
Este paquete no contiene los binarios de las imágenes. Contiene referencias públicas directas porque el entorno de generación no permitió descargar los archivos remotos al sandbox local.
